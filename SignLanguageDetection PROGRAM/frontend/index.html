Notes

  • Frontend runs in your browser (Live Server). It can optionally call a local Python server to perform predictions.
  • If you want predictions, run the Flask server provided in frontend/server.py — instructions in the frontend README.
  • Use the toggle Use Local Backend to switch between demo (no backend) and calling the server at http://127.0.0.1:5000/predict.

This frontend lives in the frontend/ folder and won't change your working Python scripts.

Press Translate to send the current frame to the server, or run in demo mode locally.
